# 👩‍💼 Employee Directory Web Interface

A responsive and interactive frontend project built with **HTML, CSS, Vanilla JavaScript**, and **Freemarker templates**. It allows viewing, adding, editing, deleting, searching, filtering, and sorting employee records — all handled in memory (no backend).

---

## 🚀 Features

- 📋 **View Employee Records**
- ➕ **Add / Edit / Delete** employees
- 🔍 **Search by name or email**
- 🎯 **Filter by First Name, Department, Role**
- 🔀 **Sort by First Name or Department**
- 📄 **Pagination**
- 📱 **Responsive UI** (mobile / tablet / desktop)
- ✅ **Form Validation** (required fields, email format)
- 💡 Clean and modular code with inline comments

---

## 🧱 Tech Stack

- ✅ HTML5 + CSS3
- ✅ JavaScript (Vanilla)
- ✅ Freemarker templating engine
- ❌ No backend or API calls — purely frontend
- 🗂️ Data stored in memory / localStorage
